# 部署

## 云服务器ECS

### DNS解析

在**购买域名的服务商平台**下添加域名解析记录，记录值指向我们**服务器的ip地址**

| 主机记录 | 记录类型 | 记录值 |
| -------- | -------- | ------ |
| @        | A        | 公网IP |
| www      | A        | 公网IP |
